# JZHU Trading Web

Vue3 交易系统前端应用

## 简介

JZHU Trading Web 是基于 Vue 3 + Vite 开发的前端应用，提供交易系统的用户界面。

## 技术栈

- Vue 3
- Vite
- JavaScript (ES6+)
- Axios

## 功能

- 行情数据展示
- K线图表
- 交易记录
- 系统设置

## 快速开始

### 安装
```bash
npm install
```

### 开发
```bash
npm run dev
```

### 构建
```bash
npm run build
```
